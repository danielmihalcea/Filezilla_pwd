Decrypteur de mot de passe FileZilla en JavaScript

Petit code JavaScript permettant de décrypter les mots de passe FileZilla stockés dans le fichier de configuration. L'avantage de l'utilisation du JavaScript est que l'utilisateur est certain que le script ne va pas concerver la trace du mot de passe décrypté et ainsi provoquer un problème de sécurité potentiel.
